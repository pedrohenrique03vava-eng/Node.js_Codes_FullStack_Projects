📝 Fullstack CRUD: User Management System
This is a complete CRUD (Create, Read, Update, Delete) application built to demonstrate fundamental fullstack development concepts. The application allows users to register their name, age, and email, displaying the data in a dynamic list with options to update or delete records.

🚀 Technologies
This project was developed using the following stack:

Frontend: HTML5 & CSS3 (Responsive UI).

Language: JavaScript (ES6+).

Backend: Node.js with Express framework.

Database: MySQL for data persistence.

🛠️ Features
Create: Form to input Name, Age, and Email.

Read: Real-time display of registered users in a list below the form.

Update: Edit button to modify existing user information.

Delete: Remove records directly from the database and UI.

📋 Prerequisites
Before you begin, ensure you have the following installed:

Node.js

MySQL (or a tool like XAMPP/WampServer)

A code editor (e.g., VS Code)
